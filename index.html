mv ഒരു പ്രത്യേക ഫയലോ ഫോൾഡറോ മറ്റൊരു ഫോൾഡർ/ഡയറക്ടറിയിലേക്ക് നീക്കാൻ ഉപയോഗിക്കുന്നു. Image.jpg നെ webdev ഡയറക്ടറിയിലേക്ക് നീക്കാൻ mv image.jpg /data /webdev പോലെ ഇത് ഉപയോഗിക്കാം .

mv {file_name} /sdcard/{file_name}

ls

പ്രവർത്തിക്കുന്ന ഫോൾഡർ/ഡയറക്ടറിയിലെ എല്ലാ ഡയറക്ടറികളും ഫയലുകളും ലിസ്റ്റ് ചെയ്യാൻ ഉപയോഗിക്കുന്ന ഒരു കമാൻഡാണ് ls . ഫയലുകൾ/ഡയറക്ടറികൾ പട്ടികപ്പെടുത്താൻ ഇത് ഉപയോഗിക്കുന്നു, അതിനാൽ നിങ്ങൾക്ക് ഒരു പ്രത്യേക ഫയലോ ഫോൾഡറോ നോക്കാവുന്നതാണ്

ls {tool_name}

clear

വ്യക്തമായ ടെർമിനൽ സ്ക്രീൻ ക്ലിയർ ഉപയോഗിക്കുന്ന ഒരു അടിസ്ഥാന തെര്മുക്സ കമാൻഡ് ആണ്. നിങ്ങൾ ധാരാളം കമാൻഡുകൾ എഴുതിയിട്ടുണ്ടെങ്കിൽ, ടെർമിനൽ സ്ക്രീൻ മാലിന്യമായി കാണപ്പെടുന്നു. ടൈപ്പ് വ്യക്തമായ സ്ക്രീൻ ക്ലിയർ നൽകുക അമർത്തുക.

pkg install

ടെർമുക്സിലെ apt കമാൻഡിനുള്ള പകരക്കാരനാണ് pkg install കമാൻഡ് . ടെർമുക്സിലേക്ക് പാക്കേജുകൾ ഇൻസ്റ്റാൾ ചെയ്യാൻ pkg ഇൻസ്റ്റാൾ കമാൻഡ് ഉപയോഗിക്കുന്നു. ഉദാഹരണത്തിന്, ടെർമാക്സിൽ പൈത്തൺ 3 ഇൻസ്റ്റാൾ ചെയ്യാൻ pkg ഇൻസ്റ്റാൾ python3 ഉപയോഗിക്കുക .

pkg install {pkg_name}

cp

ഒരു ഫയൽ മറ്റൊരു ഡയറക്ടറിയിലേക്ക് പകർത്താൻ cp കമാൻഡ് ഉപയോഗിക്കുന്നു. ഉദാഹരണം: cp image.jpg /droidmaze /webdev image.jpg ഫയൽ webdev ഡയറക്ടറിയിലേക്ക് പകർത്താൻ

cp {file_name} /sdcard/{file_name}

rm -rf

rm കമാൻഡ് ഒരു ഫയൽ / ഡയറക്ടറി നീക്കം ഉപയോഗിക്കുന്നു. rm ഉപയോഗിക്കുന്നു rm ഇമഗെ.ജ്പ്ഗ് തെര്മുക്സ നിന്നും ഇമഗെ.ജ്പ്ഗ് ഫയൽ ഇല്ലാതാക്കാൻ / നീക്കം. -R ഫ്ലാഗ് ഉപയോഗിച്ച് നിങ്ങൾക്ക് ഡയറക്ടറി ഇല്ലാതാക്കാനും കഴിയും . വാൾപേപ്പറുകൾ ഡയറക്ടറി ഇല്ലാതാക്കുന്നതിനും നീക്കം ചെയ്യുന്നതിനും ഉദാഹരണം rm -r വാൾപേപ്പറുകൾ .

rm -rf {tool_name}
